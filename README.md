# Tima Website

Site vitrine pour **Tima**, la pointeuse intelligente.

## Description

Ce projet est le site web de présentation de Tima, mettant en avant ses fonctionnalités, son excellente technique et sa promesse pour les employés et les managers.

## Installation et Lancement

1.  Installer les dépendances :
    ```bash
    npm install
    ```

2.  Lancer le serveur de développement :
    ```bash
    npm run dev
    ```

3.  Ouvrir le navigateur sur l'URL indiquée (généralement `http://localhost:5173`).

## Technologies

*   HTML/CSS/JavaScript
*   Vite
